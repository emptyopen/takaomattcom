# Static assets

Drop your portrait here as **`me.jpg`** (square works best — it's masked to a
circle). It will appear in the homepage hero automatically. Until then, the
avatar shows "MT" initials as a fallback.

You can also use a different format by updating the `src` in
`src/components/avatar.tsx` (e.g. `/me.png` or `/me.webp`).
